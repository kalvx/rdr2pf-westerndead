Server Commands

Confirmed administrative, development, encounter, AI, zombie, and FXServer resource commands used while developing Western Dead.

AI and Encounter Commands

Run these commands from the in-game chat or F8 console while testing custom encounters and spawned AI.

Name Command Where Notes
Spawn Encounter Camp spawncamp In-game / F8 Spawns the custom AI encounter camp for testing.
Clear Encounter AI clearai In-game / F8 Deletes AI spawned by the primary RDR2PF AI encounter system.
Clear Secondary AI clearai2 In-game / F8 Deletes AI spawned by the secondary encounter or faction system.

RDR2PF Resource Commands

Run these from the FXServer console or txAdmin console. Do not place a slash before server-console commands.

Resource Command Purpose
AI and Encounters restart rdr2pf_ai Reloads the custom AI and encounter resource after script changes.
Zombies restart rdr2pf_zombies Reloads zombie models, spawning, management, and AI behavior.
Pause Menu restart rdr2pf_pausemenu Reloads the custom Western Dead pause menu.
Unconfirmed crowd command: restart rdr2pf_crowd is not documented because no confirmed resource named rdr2pf_crowd has been found. The crowd restart may instead be a registered command inside another resource.

FXServer Resource Management

General commands used to discover, start, reload, and stop server resources.

Command Purpose Example
refresh Rescans the resources directories and discovers newly added or changed resource manifests. It does not restart resources. refresh
ensure <resource> Starts a stopped resource. If it is already running, FXServer reloads it. ensure rdr2pf_zombies
restart <resource> Stops and starts an existing running resource. restart rdr2pf_ai
stop <resource> Stops a running resource without starting it again. stop rdr2pf_zombies
start <resource> Starts a resource that is currently stopped. start rdr2pf_zombies

Common Development Sequences

Reload an Existing Resource

restart rdr2pf_ai

Discover and Start a Newly Added Resource

refresh
ensure resource_name

Reload Zombie Development Changes

restart rdr2pf_zombies

Clear Encounter AI Before Reloading

clearai
clearai2

Command Location Notes

Location Command Format
In-game chat May require a leading slash, such as /clearai.
RedM F8 console Usually entered without the slash, such as clearai.
FXServer or txAdmin console Resource commands are entered without a slash, such as restart rdr2pf_zombies.
