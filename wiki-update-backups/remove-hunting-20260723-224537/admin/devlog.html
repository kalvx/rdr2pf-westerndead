Development Journal

Development Log

Implementation history, design decisions, discoveries, and follow-up work.

July 23, 2026 - Jobs and Profession Systems Documented

Added a permanent Jobs & Professions page covering the systems currently enabled, configured, tested, or under repair.

  • Documented hunting/butcher, fishing, herbs/farming, crafting/cooking, lumberjack, mining, doctor, outlaw, and NPC-looting systems.
  • Corrected the lumberjack tool reference from lumberaxe to the configured database item hatchet.
  • Recorded the mining mismatch between configured goldnugget and database item gold_nugget for later repair and testing.
  • Separated configured resources from fully tested gameplay loops so the wiki does not claim unfinished behavior is working.

July 23, 2026 - 997-Item Database Registry Preserved

The MariaDB items table was confirmed as the authoritative inventory registry. A 997-item ID snapshot was preserved in data/item-ids.txt, and a permanent crafting-validation procedure was added before repairing broken recipes.

  • Recipes must use the exact value from items.item.
  • egg and eggs were confirmed as separate existing item IDs.
  • pork and consumable_breakfast were confirmed present.
  • Future recipe repairs will be compared against the database rather than guessed from labels.

July 14, 2026 - Database-Confirmed Save & Quit

A custom Save & Quit flow was completed for the RDR2PF pause menu.

  • The client sends current coordinates and heading to the server.
  • The server resolves the requesting player's active VORP character.
  • The full character record is saved through SaveUser() and SaveCharacterInDb().
  • The MariaDB update is awaited directly.
  • The player is disconnected only after the database reports success.
  • The system operates independently for every connected player.

July 14, 2026 - Rockstar Menu Emergency Fallback

The Open Rockstar Menu button now closes the custom NUI, releases focus, and triggers the alternate frontend pause control. This provides a recovery path if the custom menu or another NUI becomes unusable.

July 14, 2026 - Character Journal Framework

The Character button was enabled and a separate Character Journal panel was added. Back navigation is working. Live character values will be read from LocalPlayer.state.Character rather than queried from MariaDB whenever the page is opened.

Design Direction

The finished pause menu is intended to replace the normal Rockstar pause flow after it reaches feature parity. Planned presentation includes subtle smoke, ash, lantern flicker, journal styling, smoother transitions, character information, horse information, hometown, playtime, infection status, reputation, skills, statistics, and achievements.
