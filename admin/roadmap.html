PRIVATE FRONTIER (RDR2PF) — ROADMAP SUMMARY

CURRENT PHASE
Gameplay validation and documentation.

FOUNDATION
- FXServer / txAdmin startup standardized.
- MariaDB and VORP operational.
- Characters, coordinates, horses, weapons, and inventory largely working.

VALIDATION
- Hunting, skinning, cooking, crafting, banking, fishing, wagons, housing, and trading.
- Record every confirmed item ID, resource behavior, bug, and fix in this wiki.

LIVING MASTERY
- Actions earn hidden mastery points.
- Highest mastery becomes current title.
- Cook, Hunter, Peddler, Exterminator, Crook, and Murderer are current title directions.
- Repeated actions improve related benefits.

SURVIVAL WORLD
- Limited infected population.
- Infected remain meaningful rather than becoming endless background noise.
- Living NPC infection is a future direction.
- Major settlements become high-risk, high-value destinations.

Custom Pause Menu and Character Journal

STATUS: ACTIVE DEVELOPMENT

COMPLETED
- Custom RDR2PF pause menu resource created.
- Resume button operational.
- Map button opens the Rockstar map.
- Save & Quit performs a full VORP character save.
- Save & Quit waits for MariaDB confirmation before disconnecting.
- Save flow is multiplayer-safe and resolves each character from the requesting player's server source.
- Saved coordinates and money persistence verified after reconnect.
- Emergency Open Rockstar Menu fallback operational.
- Character button and Character Journal panel framework created.

IN PROGRESS
- Populate the Character Journal with live VORP character data.
- Display name, nickname, occupation, money, gold, XP, age, gender, description, character ID, and server ID.
- Add horse, hometown, playtime, and server information.
- Add a normal Disconnect option separate from Save & Quit.
- Build the Settings page.
- Improve handling when another NUI, such as inventory, becomes stuck.

VISUAL POLISH
- Subtle smoke behind the menu.
- Drifting ash particles.
- Lantern or firelight flicker.
- Journal-inspired Western styling.
- Smooth page transitions.
- Character portrait area.

FUTURE
- Replace the normal Escape menu only after feature parity is complete.
- Keep Open Rockstar Menu as an emergency fallback.
- Add hometown, infection status, reputation, skills, statistics, achievements, and horse information.
